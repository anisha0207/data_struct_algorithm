PA 9:
TreeMap:
- first(): returns a key-value pair with the lowest key in this tree map, or null if the tree map is empty.
- last(): returns a key-value pair associated with the highest key in this tree map, or null if the tree map is empty.
- ceiling(K k): returns a key-value pair associated with the least key greater than or equal to k, or null if the tree is empty or k is greater than all the keys in the tree.
- floor(K k): returns a key-value pair associated with the greatest key less than or equal to k, or null if the tree is empty or k is less than all the keys in the tree.
- keys(): returns a collection of the keys in the tree using the inorder traversal.
- values(): returns a collection of the values in the tree using the inorder traversal.
